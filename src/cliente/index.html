Ejemplos Keynua

  • 01. Keynua - Contratos - uiCallback
    Ejemplo de integración de contratos utilizando el parametro uiCallback. Este parametro permite recibir eventos en tiempo real del widget de Keynua, mostrando los eventos en la misma página. Permite monitorear en tiempo real el progreso de la firma de un contrato.
  • 02. Keynua - Identificaciones - uiCallback
    Ejemplo de integración de identificaciones utilizando el parametro uiCallback. Similar al ejemplo de contratos, pero enfocado en el proceso de identificación de usuarios. Permite monitorear en tiempo real el progreso de la identificación.
  • 03. Keynua - Identificaciones - eventURL
    Ejemplo de integración de identificaciones redirigiendo a una URL con los parametros eventDoneURL y eventErrorURL. Este metodo es ideal cuando no se requiere monitorear en tiempo real el progreso de la identificación y que se firme dentro de la pagina de keynua redirigiendo en caso de error o de exito (Pulsando el boton de finalizar).
  • 04. Keynua - Contratos - theme
    Ejemplo de integración de contratos con personalización de tema. Puede editar las variables keynuaTheme y keynuaPrimitives para cambiar el estilo del Widget. También puede modificarlos de manera interactiva por la web.
    NOTA: los estilos son generados a partir de la configuracion de la organizacion en keynua, que genera los "primitives" y estos generan el "theme".
